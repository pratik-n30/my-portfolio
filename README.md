# Pratik Nawale - Portfolio

A high-performance personal portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS**. 

This project uses a custom **File-System Based CMS**, meaning it reads projects and timeline data directly from local files (JSON/Markdown) without needing a database.

* **Framework:** Next.js 15 (App Router)
* **Styling:** Tailwind CSS (Dark Mode)
* **Language:** TypeScript
* **Architecture:** Static Site Generation (SSG) for speed

## üìÇ Features

* **"VS Code Style" Project Viewer:** Browse code and READMEs directly in the browser.
* **Dynamic Routing:** Automatically generates pages for new projects added to the `content/` folder.
* **Zero-Database:** Content is managed via a designated "safe zone" folder.
